# dl-cli

Download videos & audio from YouTube, TikTok, Instagram, Twitter/X, Spotify and 1000+ sites.

Lightweight CLI media downloader with progress bar.

## Getting Started

### Step 1: Install yt-dlp

```bash
pip install yt-dlp
```

### Step 2: Install ffmpeg

Download from [ffmpeg.org](https://ffmpeg.org/) and add to your PATH.

### Step 3: Download dl-cli

Download `dl-cli.exe` from [Releases](https://github.com/imaan-jaman/dl-cli/releases) and put it anywhere.

### Step 4: Run

```
dl-cli                              Interactive menu
dl-cli <url>                        Download video
dl-cli <url> -a                     Download audio (MP3)
dl-cli <url> -q 2                   Quality: 1=best 2=1080p 3=720p 4=480p
dl-cli <url> -o D:\Downloads        Save to directory
```

## Platforms

YouTube, TikTok, Instagram, Twitter/X, Facebook, Twitch, Reddit, SoundCloud, Vimeo, Bilibili, Spotify, Dailymotion, and 1000+ more sites.
